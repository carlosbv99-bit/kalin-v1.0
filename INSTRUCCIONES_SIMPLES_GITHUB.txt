================================================================================
KALIN v3.0 - INSTRUCCIONES SIMPLES PARA SUBIR A GITHUB
================================================================================

OPCION 1: LA MAS FACIL (Windows)
---------------------------------
1. Haz doble clic en el archivo: prepare_github.bat
2. Espera a que termine
3. Sigue las instrucciones que aparecen en pantalla

OPCION 2: CON POWERSHELL (Windows - Mas avanzado)
---------------------------------------------------
1. Abre PowerShell en esta carpeta
2. Escribe: .\prepare_for_github.ps1
3. Espera a que termine
4. Sigue las instrucciones

OPCION 3: MANUAL (Todos los sistemas)
---------------------------------------
Abre una terminal y ejecuta estos comandos:

python verify_github_ready.py
python auto_prepare_github.py
git add .
git commit -m "Clean project structure"
git push origin main

================================================================================
LO QUE NO SE SUBIRA A GITHUB (PROTEGIDO)
================================================================================

Estos archivos estan protegidos y NO se subiran:

X .env                        -> Tus API keys (MUY IMPORTANTE)
X sessions/                   -> Tus conversaciones
X experience_memory/          -> Tu memoria personal
X logs/                       -> Tus registros
X __pycache__/                -> Archivos temporales de Python
X backups/                    -> Copias de seguridad
X .gradle/, .idea/            -> Configuracion del IDE

================================================================================
LO QUE SI SE SUBIRA A GITHUB
================================================================================

Estos archivos SI se subiran (son seguros):

V agent/                      -> Codigo del agente AI
V app/                        -> Aplicacion Android
V static/, templates/         -> Pagina web
V tests/                      -> Pruebas
V README.md                   -> Documentacion
V .env.example                -> Plantilla (sin tus claves reales)

================================================================================
VERIFICACION RAPIDA
================================================================================

Antes de subir, verifica con este comando:

git status

Deberias ver tus archivos de codigo, pero NO deberias ver:
- .env
- sessions/
- logs/
- __pycache__/

Si ves alguno de esos, algo esta mal. Ejecuta: python verify_github_ready.py

================================================================================
COMANDOS UTILES
================================================================================

Ver estado de Git:
git status

Ver que archivos se ignoran:
git status --ignored

Ver que archivos se subiran:
git ls-files

Agregar todos los archivos:
git add .

Crear commit:
git commit -m "Tu mensaje aqui"

Subir a GitHub:
git push origin main

================================================================================
PROBLEMAS COMUNES Y SOLUCIONES
================================================================================

PROBLEMA: El archivo .env aparece en git status
SOLUCION: Ejecuta estos comandos:
    git rm --cached .env
    git commit -m "Remove .env"

PROBLEMA: No estoy seguro si es seguro subir
SOLUCION: Ejecuta: python verify_github_ready.py

PROBLEMA: Quiero ver que archivos se excluiran
SOLUCION: Ejecuta: git status --ignored
