================================================================
           KALIN AI - COMO INICIAR EL PROGRAMA
================================================================

OPCION 1: MAS FACIL (Recomendado para principiantes)
----------------------------------------------------------------
1. Busca el archivo "Iniciar_Kalin.bat" en esta carpeta
2. Haz DOBLE CLICK en el archivo
3. Espera a que se abra el navegador
4. ¡Listo! Kalin está funcionando

Para cerrar Kalin:
- Cierra la ventana negra que se abrió
- O presiona Ctrl+C en esa ventana


OPCION 2: USANDO POWERSHELL (Windows moderno)
----------------------------------------------------------------
1. Haz click derecho en "Iniciar_Kalin.ps1"
2. Selecciona "Ejecutar con PowerShell"
3. Si aparece un mensaje de seguridad, selecciona "Sí"
4. Espera a que se abra el navegador


OPCION 3: MANUAL (Para usuarios avanzados)
----------------------------------------------------------------
1. Abre Terminal o PowerShell en esta carpeta
2. Escribe: python run.py
3. Presiona Enter
4. Abre tu navegador en: http://localhost:5000


PRIMERA VEZ QUE USAS KALIN?
----------------------------------------------------------------
Si es la primera vez que usas Kalin:

1. Asegúrate de tener Python instalado
   - Descarga desde: https://www.python.org/downloads/
   - Marca la opción "Add Python to PATH" durante instalación

2. Abre Kalin usando la OPCIÓN 1 (doble click en .bat)

3. En el menú lateral (botón ☰), haz click en:
   "🚀 Instalar Kalin (Automático)"

4. Espera a que termine la instalación (puede tardar 10-30 min)

5. Reinicia Kalin y ¡comienza a usarlo!


PROBLEMAS COMUNES
----------------------------------------------------------------
Problema: "Python no está reconocido"
Solución: Instala Python y marca "Add to PATH"

Problema: La ventana se cierra inmediatamente
Solución: Verifica que Python esté instalado correctamente

Problema: Error al abrir el navegador
Solución: Abre manualmente http://localhost:5000

Problema: No funciona el botón del menú
Solución: Presiona Ctrl+F5 para recargar la página


NECESITAS AYUDA?
----------------------------------------------------------------
- Revisa la documentación en README.md
- Verifica los logs en la carpeta "logs"
- Asegúrate de tener Ollama instalado para modelos locales
