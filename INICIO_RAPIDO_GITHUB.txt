================================================================================
KALIN v3.0 - INICIO RÁPIDO PARA GITHUB
================================================================================

¿Quieres subir tu proyecto a GitHub? ¡Es muy fácil!

================================================================================
LA FORMA MÁS FÁCIL (1 SOLO COMANDO)
================================================================================

Windows (PowerShell):
    .\auto_github_prep.ps1

Todos los sistemas (Python):
    python auto_github_prep.py

¡Eso es todo! El script hace TODO automáticamente.

================================================================================
¿QUÉ PASA DESPUÉS DE EJECUTAR EL SCRIPT?
================================================================================

El script hará automáticamente:
  ✓ Verificará tu proyecto
  ✓ Limpiará archivos temporales
  ✓ Protegerá archivos sensibles
  ✓ Generará un reporte

Al final te dirá exactamente qué hacer.

Si usas PowerShell, incluso te preguntará si quieres hacer commit y push
automáticamente. ¡Solo responde "s" y listo!

================================================================================
SI PREFIERES HACERLO MANUALMENTE
================================================================================

git add .
git status                      ← Verifica que no haya .env
git commit -m "Ready for GitHub"
git push origin main

================================================================================
ARCHIVOS QUE NO SE SUBIRÁN (PROTEGIDOS)
================================================================================

Estos archivos están protegidos y NO se subirán a GitHub:

  X .env                    → Tus API keys (MUY IMPORTANTE)
  X sessions/               → Tus conversaciones
  X experience_memory/      → Tu memoria personal
  X logs/                   → Tus registros
  X __pycache__/            → Archivos temporales
  X backups/                → Copias de seguridad

================================================================================
ARCHIVOS QUE SÍ SE SUBIRÁN
================================================================================

  V agent/                  → Código del agente AI
  V app/                    → Aplicación Android
  V static/, templates/     → Página web
  V tests/                  → Pruebas
  V README.md               → Documentación
  V .env.example            → Plantilla (sin tus claves)

================================================================================
¿NECESITAS MÁS INFORMACIÓN?
================================================================================

Lee estos archivos en orden:

1. AUTO_GITHUB_INSTRUCTIONS.txt   → Instrucciones de scripts automáticos
2. SCRIPTS_AUTOMATICOS.md         → Detalles de scripts automáticos
3. README_GITHUB.md               → Guía principal completa
4. INSTRUCCIONES_SIMPLES_GITHUB.txt → Instrucciones básicas

================================================================================
VERIFICACIÓN RÁPIDA
================================================================================

Antes de subir, verifica con:

    git status

Deberías ver tus archivos de código, pero NO deberías ver:
  - .env
  - sessions/
  - logs/
  - __pycache__/

Si ves alguno de esos, ejecuta: python auto_github_prep.py

================================================================================
PROBLEMAS COMUNES
================================================================================

P: ¿No puedo ejecutar el script de PowerShell?
R: Ejecuta primero: Set-ExecutionPolicy -Scope CurrentUser RemoteSigned

P: ¿Python no está instalado?
R: Usa el script de PowerShell o instala Python desde python.org

P: ¿Git no está instalado?
R: Instala Git desde git-scm.com

P: ¿No estoy seguro si es seguro subir?
R: Ejecuta: python auto_github_prep.py (te dirá si hay problemas)

================================================================================
RESUMEN EN 3 PASOS
================================================================================

1. Ejecuta: python auto_github_prep.py
   O: .\auto_github_prep.ps1

2. Lee el resultado en pantalla

3. Si dice "TODO LISTO", ejecuta:
   git add .
   git commit -m "Ready"
   git push origin main

¡Listo! ✅
