## ⚡ Quick Start

```bash
# Clonar repositorio
git clone https://github.com/carlosbv99/kalin.git
cd kalin

# Instalar dependencias
pip install -r requirements.txt

# Configurar Ollama
ollama pull deepseek-coder:latest
ollama serve

# Ejecutar Kalin
python run.py
```

Accede a: **http://localhost:5000**
