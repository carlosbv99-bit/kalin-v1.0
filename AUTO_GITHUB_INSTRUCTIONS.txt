PROBLEMAS COMUNES
================================================================================

PROBLEMA: El script dice que .env está en Git
SOLUCIÓN: El script lo eliminará automáticamente, o ejecuta:

PROBLEMA: No puedo ejecutar el script de PowerShell
SOLUCIÓN: Ejecuta primero:

PROBLEMA: Python no está instalado
SOLUCIÓN: Usa el script de PowerShell o instala Python desde python.org

PROBLEMA: Git no está instalado
SOLUCIÓN: Instala Git desde git-scm.com
